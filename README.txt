Programs all written in matlab.

All files needed for the particle filter based localisation program, using an NXT Mindstorms robot (2 motors for wheels, ultrasound mounted on a third motor).
Requires various NXT software to be installed, along with the libraries provided in botsimlib.

original_localise.m provides a functioning localisation system using a simulated robot.

Note that botsimlib must be added to the path when attempting to run any of these files.
